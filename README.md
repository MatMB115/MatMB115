### Projetos em Destaque

#### Pesquisa e Inteligência Artificial
* **[Deepfake Detection](https://arxiv.org/abs/2504.02900)** — Estudo comparativo de 7 modelos para detecção de mídias falsificadas nos datasets DeepSpeak e WildDeepfake. [[Repo](https://github.com/MatMB115/tfg-deepfake-detection)]
* **[Minesweeper MVC](https://github.com/MatMB115/minesweeper_mvc_patterns_com221)** — Implementação de Campo Minado em Python e Qt5 utilizando padrões de projeto MVC e Strategy para desacoplamento de lógica e interface.

#### Backend e Engenharia de Dados
* **[API AD-HOC IGDB](https://github.com/MatMB115/ad_hoc_igdb_back_end)** — Desenvolvimento de API para geração de relatórios dinâmicos, permitindo execução de queries customizadas no banco de dados IGDB.
* **[Script de Carga IGDB](https://github.com/MatMB115/script_carga_igdb)** — Pipeline de ETL para consumo da API IGDB utilizando Flask, SQLAlchemy e PostgreSQL.

* **[CountingSort High Performance](https://github.com/MatMB115/CountingSort-e-E-CountingSort)** — Implementação otimizada de algoritmos de ordenação linear em linguagem C.
